Build a QDigest from an integer vector.

Allocates and initializes a QDigest sized for upper_bound, then inserts the size values from a into it, returning the populated digest. While in the message-passing set-up the buffer provided correspond with the local buffer of the node, in the shared-memory set up, a pointer to the local buffer is passed along with start and end, which indicate the portion of the buffer being processed by the thread.

Parameters
aPointer to the input array of integers.
startZero-based index identifying the first element of the segment to process.
endZero-based index identifying the boundary one past the last element of the segment (exclusive).
sizeNumber of elements in a.
upper_boundMaximum value expected in a (used to size the digest).
Returns
Newly allocated QDigest containing all values from a.

Fills a buffer with pseudo-random integers for a given thread.

Seeds the C RNG with rank so every process chosen for the generation (default case is the master thread) creates a reproducible sequence, then writes n values between the given lb and ub (lower bound and upper bound) (inclusive) into data.

Parameters
thread_idCalling thread ID; used both for RNG seeding and to decorrelate buffers.
dataPointer to the buffer that will receive n integers.
nNumber of elements to generate.
lblower bound of data.
upupper bound of data.
